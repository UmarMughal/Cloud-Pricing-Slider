A) HTML Structure - top

Please see the index.html and use line 49-150 to use slider for any bootstrap based theme/template. You can also use this slider in any other framework, just adjust the div as per requirements of framework markups or simple html as you can see the slider is divided in two columns.

B) CSS Files and Structure - top

Include the following css

assets/css/slider.min.css (Main Slider css)
assets/css/jquery-ui.min.css

To customize look and feel make changes in CSS. Uncompressd CSS and "Less" also included assets/css/ folder.

Note: JQuery UI css is not required if your template/theme already has within.

C) JavaScript - top

Include the following scripts

assets/js/jquery-1.11.0.min.js
assets/js/jquery-ui.min.js
assets/js/qs.slider.js (Main Slider script)
assets/js/qs.slider.init.js (Initial Script for available custom options)

Note: jQuery and JQuery UI scripts are not required if your template/theme already has within.

Please open assets/js/qs.slider.init.js in any of text editor (recommended Notepad ++) to change slider settings i.e. Pricing, link etc..

To configure preset buttons selected sliders values, For now you need to edit main qs.slider.js
(chnage values in lines 204 - 208) i.e.
xs: { cpu: "3", ram: "4", hdd: "10" }, etc..
